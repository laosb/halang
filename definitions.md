## 函数

函数的定义类似 ES6 语法。

```halang
注释 常规方式
函数 函数名（参数一，参数二）{
  语句；
  返回 返回值；
}

注释 箭头函数，直接返回值
常量 函数 = （参数一，参数二）=》返回值；

注释 箭头函数，返回值是对象
常量 函数 = （参数一，参数二）=》（{ 键：值 }）；

注释 箭头函数，单参数
常量 函数 = 参数 =》 返回值；

注释 箭头函数，不直接返回
常量 函数 = 参数 =》 {
  语句；
  返回 返回值；
}
```
