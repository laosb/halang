# HaLang

```halang
从 「./毛泽东思想」 导入 毛思；
从 「./邓小平理论」 导入 邓理；
从 「./三个代表重要思想」 导入 三代；
从 「./科学发展观」 导入 科观；

常量 中特社义理论体系 = {
    毛思，
    邓理，
    三代，
    科观，
}；

导出 默认 中特社义理论体系；
```

基于 JavaScript 的中文语言。
